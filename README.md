# 😵 Kariyer Krizi Simülatörü

Yeni mezunların kariyer yolculuğunu mizahi şekilde analiz eden interaktif web uygulaması.

## Özellikler

- Dinamik soru sistemi
- Skor hesaplama
- Sonuç ekranı
- İlerleme çubuğu
- Yeniden oynama özelliği

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript

## Amaç

Bu proje web geliştirme öğrenme sürecinde oluşturulmuştur.
